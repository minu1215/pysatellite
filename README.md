# use

### class Presenter
- addPresenter(name) : 발표자 추가
- print(index) : index에 해당하는 발표자 출력
- randomIndex : 발표자 숫자에 맞는 random Index 반환

### class StringPresenter
- constructor(Array) : 발표자 배열 생성시 적용

### how to use
```
$ npm install -g kaz1301
$ kaz1301 presenter1 presenter2 ...
```
  
### example

$ npm install -g kaz1301
